Web Straping

Qu'est ce que le scraping de données ?


Le web scraping, également connu sous le nom d’extraction de données Web, est le processus de récupération ou de «raclage» des données (crawling) d’un site Web. Contrairement au processus banal et insensé d’extraction manuelle des données, le Web scraping utilise une automatisation intelligente pour récupérer des centaines, des millions, voire des milliards de points de données sur Internet. Ces informations sont collectées puis exportées dans un format plus utile pour l’utilisateur. Que ce soit une feuille de calcul ou une API. Les entreprises ou les sites de comparaison des prix en ligne utilisent les données extraites sur les sites web des différents concurrents pour fournir un comparatif sur un produit.

Fonctionnement des web scrapers

  • Tout d’abord, le web scraper recevra une ou plusieurs URL à charger avant le scraping.
  • Ensuite, le Scraper extraira toutes les données de la page ou des données spécifiques sélectionnées par l’utilisateur avant l’exécution du projet.
  • Enfin, le Web Scraper affichera toutes les données collectées dans un format plus utile à l’utilisateur. Les web scrapers peuvent alors stocker les données dans un fichier csv ou excel ou dans un fichier json qui peut être utilisé pour une API. Ces informations peuvent être également stockée dans les entrepots de données.

A quoi sert le web scraper ?

  • Suivi des prix
  • Etude de marché
  • Surveillance des actualités et du contenu
  • L'immobilier

Les outils de Web scraping :

Les langages de programmation les plus utilisés pour le web scraping sont python et javascript. Les librairies python les plus populaires pour le web scraping sont: scrapy, beautifulsoup4, Selenium.

Les extensions de web scraping ou des logiciels de web scraping servent également aux web scrapers. D'un coté, les extentions de web scraping qui sont simples à exécuter et peuvent intégrées directement dans un navigateur et d'un autre coté, les logiciels de web scraping qui peuvent être téléchargés et installés sur l'ordinateur.

L'on dispose aussi des proxys, qui servent à interdir un ou plusieurs sites de vous retracer ou également de vous bloquer l'accès à leurs services. Nous pouvons noter pour exemple le proxy luminati.



Les pratiques légales et illégales du web scraping.


Bien que le web scraping soit une technique très efficace il comporte éventuellement des risques et son utilisation, pouvant dans certains cas, être pis pour une arnaque, il comportera toujours certaines failles à ne pas dépasser. Sachez qu’il est tout à fait légal de faire une collecte d’information auprès de sites internet, en fonction bien sûr de leurs conditions générales de vente et de leur contrat avec leurs clients.

Un avantage pour ses utilisateurs, mais qui peut cependant être transformé en une arnaque pour des personnes malintentionnées. En effet, en dehors des adresses email, il est possible d’obtenir plusieurs informations concernant un client comme son nom et prénom, sa date de naissance, mais également les informations sur sa carte bancaire. Le web scraping a donc une double face, mais il faut juste l’utiliser à bon escient.
